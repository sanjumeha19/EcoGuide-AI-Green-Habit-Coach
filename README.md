## Project Overview
EcoGuide AI is a conversational chatbot designed to help users make eco-friendly daily decisions. It provides simple sustainability advice based on user input such as weather or daily activities.

## Problem Statement
People often make daily choices without considering their environmental impact. This chatbot helps bridge the gap between awareness and action by offering timely, practical suggestions.

## AI Approach
- Agent-based AI logic
- LLM-powered conversational responses
- Built using Voiceflow (no-code platform)

## Tools Used

- Voiceflow
- Agentic AI
- Web-based Chat Interface
